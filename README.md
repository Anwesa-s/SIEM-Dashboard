# SIEM Dashboard

A full-stack Security Information and Event Management (SIEM) dashboard built using FastAPI and React.

## Features

- Log ingestion
- Event normalization
- SQLite database storage
- Event querying
- Brute-force attack detection
- Alert generation
- React dashboard visualization

## Tech Stack

### Backend
- FastAPI
- SQLAlchemy
- SQLite

### Frontend
- React
- Vite
- Axios
- Recharts

## Detection Rules

- Brute-force detection:
  - 5 failed logins from same IP
